# README

### Required information to run the project in local
1. Used Ruby and Rails version:
```
ruby '2.5.7'
'rails', '~> 6.1.6', '>= 6.1.6.1'
```
2. After cloning, please run following command to install used packages

```
bundle install
yarn install
```
3. To enable caching in development mode
```
rails dev:cache
```
